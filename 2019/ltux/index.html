Talk Summary

What is needed to begin an accessibility program

  • Someone who cares and is willing to ask questions to get the conversation started

What is needed to run an accessibility program?

  • Executive buy-in
    • In other words: accessibility needs to be a requirement because teams won't do things that aren't required
    • In other words: some accessibility issues need to be able to block launches
  • At least two accessibility checkpoints
    • At the end of design: Don't waste time coding inaccessible designs
    • Before launch: Don't ship an incomplete product
  • Expert
    • Create policies
    • Enforce accessibility guidelines and best practices
    • Educate and consult with product teams
    • Create a culture of accessibility
    • Help teams prioritize accessibility issues and determine which will block launches
