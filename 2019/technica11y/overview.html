For the last few years I've given a talk exploring the exceptions to WCAG Success Criteria. While covering color-related success criteria, I show these examples:

  • Somewhere in this sentence is a link that fails 1.4.1.

  • Somewhere in this sentence is a link that passes 1.4.1.

I would also sometimes say things like...

  • I kind of hate the color alone SC
  • The color contrast algorithm doesn't always work...but it's what we have
  • The only difference between nothing and text is color in the shape of letters
  • I'm going to make a talk about how much I hate the "color alone guideline"

I've also encountered many things from wonderful people that try to meet the letter of WCAG, but not necessarily the spirit

This is not a talk about gotchas when using color and how to get over them

Think of it as more of a warning of some things people might want to try to get away with so you can be prepared to say "no."
