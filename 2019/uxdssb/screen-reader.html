Basic Gestures

  • Swipe right: Next item
  • Swipe left: Previous item
  • Tap: Select an item
  • Double-tap: Activate item
  • Tap and drag: Explore by touch

iOS

  1. Go to Settings > General > Accessibility > Accessibility Shortcut
  2. Select VoiceOVer
  3. Activate Accessibility Shortcut to toggle on/off
    • Pre-iPhone X: Triple-click Home button
    • iPhone X or later: Triple-click Side button

Android

  1. Go to Settings > Accessibility > Volume key shortcut (or Accessibility shortcut)
  2. Turn on Volume key shortcut
  3. Go to Shortcut service
  4. Select TalkBack
  5. Activate Volume key shortcut (or Accessibility Shortcut) to toggle on/off
    • Press both volume buttons for 3 seconds
