How I Learned to Stop Worrying and Ignore WCAG

1.4.5 Images of Text (AA)

Summary of Images of Text

  • Use text to show text
  • Don't use images to show text

    Key Terms in Images of Text

    visually customized
    The font, size, color, and background of the text can be set
    essential
    useless without it

    Exceptions to Images of Text

    Customizable
    The image of text can be customized by the user
    Essential
    The image is essential

    Examples

    • Logos with text
    • Branding
    • Representation of a font family
    • Representation of a historical letter or document
    • Font not widely available
    • Customizable image text
    • Dynamically generated images that can be modified
