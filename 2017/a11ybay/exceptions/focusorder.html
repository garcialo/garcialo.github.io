How I Learned to Stop Worrying and Ignore WCAG

2.4.3 Focus Order (A)

Summary of Focus Order

Tab focus order through a page should make sense

Key Terms in Focus Order

navigated sequentially
navigated from one item to the next with a keyboard

Exception to Focus Order

Only applies if content/component requires a sequential order.

  • If the content doesn't require a sequential order
  • Page components that don't require a sequential order

Examples

  • Links in a tag cloud
  • Mega menu
