How I Learned to Stop Worrying and Ignore WCAG

Success Criteria with Media Alternative Exception

  • 1.2.1 Audio-only and Video-only (Prerecorded) (A)
  • 1.2.2 Captions (Prerecorded) (A)
  • 1.2.3 Audio Description or Media Alternative (Prerecorded) (A)

Media Success Criteria Summary

All information portrayed by media needs to be presented in another way.

Media Alternative Exception

If the media itself serves as an alternative, then it is exempt from needing an alternative.

Examples

  • Video of Sign Language
  • Audio description
