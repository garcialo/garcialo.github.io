How I Learned to Stop Worrying and Ignore WCAG

Success Criteria with Link Purpose Exception

  • 2.4.4 Link Purpose (In Context) (A)
  • 2.4.9 Link Purpose (Link Only) (AAA)

Summary of Link Purpose Success Criteria

2.4.4
Users should be able to tell where a link goes based on the link text and its context
2.4.9
Users should be able to tell where a link goes based on the link text alone

Link Purpose Exception

If no one would know where a link goes without going to it, WCAG does not apply

Examples

One of the notable exports is guava

Where does the guava link go?

  • Definition of guava?
  • Chart of guava exports?
  • Photo of guava being harvested?
