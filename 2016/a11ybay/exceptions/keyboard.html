How I Learned to Stop Worrying and Ignore WCAG

2.1.1 Keyboard (A)

Summary of Keyboard

  • All actions on a page can be done with a keyboard

Exceptions to Keyboard

If an action depends on a path of user movement (i.e. more than just endpoints)

Examples

  • Brush strokes in a painting program
  • Flight training simulator
