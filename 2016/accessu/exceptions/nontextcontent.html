Exploring Exceptions in WCAG Success Criteria

1.1.1 Non-text Content (A)

Summary of Non-Text Content

  • Text is easy for assistive technology to process
  • For Useful non-text content, you need to provide a text version
  • The text version needs to serve the same purpose

Jargon in Non-Text Content

text
representation of human language
text alternative
text programmaticaly associated with non-text content OR referred to by text programmatically associated with non-text content
name
text that a program can use to identify a component; can be hidden
pure decoration
only aesthetic purpose; presents no information; has no function
specific sensory experience
not purely for decoration. no important information; serves no important function
