How I Learned to Stop Worrying and Ignore WCAG

Success Criteria with Not a Change of Context Exception

  • 3.2.1 On Focus (A)
  • 3.2.2 On Input (A)

Summary of On Focus and On Input

The page doesn't change On Focus or On Input or:

Don't just do stuff that isn't expected On Focus or On Input

Key Terms in Link Purpose (In Context)

change of context
Changing the:
  • user agent
  • viewport
  • focus
  • content that changes the meaning of the page
...in a way that can disorient the user.

'Not a Change of Context' Exception

On Focus and On Input only apply if you are changing the context. Not all changes of content are changes of context.

Examples

  • Tooltips
  • Changing content after current user position
  • Moving focus in a way that makes sense
  • Set expectations of behaviors for users
