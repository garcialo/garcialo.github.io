How I Learned to Stop Worrying and Ignore WCAG

2.1.2 No Keyboard Trap (A)

Summary of No Keyboard Trap

  • Focus doesn't get trapped while tabbing through a page.

Exceptions to No Keyboard Trap

  • Modal dialogs
  • You can escape the trap using standard keyboard commands
    • unmodified arrow keys
    • unmodified tab key
    • standard exit methods
      • If an unstandard method is used, user must be advised of the method
