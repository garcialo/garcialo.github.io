A11y Tools

Enterprise Scanning Tools

Not going to discuss a specific tool

I've used or interacted with the following tools to some extent...but it's been at least a year

  • WorldSpace Comply (Deque)
  • AMP (Level Access)
  • Siteimprove
  • Rational Policy Tester (IBM)

Observations

Crawling

  • Start at a page, crawl based on site/subsite patterns
  • Work best for pages not behind authentication
  • Work okay for pages behind authentication, but don't seem to always work
  • Not great for flows

Usefulness of Issues Found

  • Can find a lot of issues
  • Can also find too many issues
  • If you fix most issues, high scores can be misleading
