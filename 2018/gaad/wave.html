A11y Tools

WAVE

Details and Demo

  • Get WAVE for Chrome
  • Get WAVE for Firefox
  • Visit WAVE Online

Pros

  • Really simple to use
  • Easy to share a page report with others
  • Goes beyond WCAG
    • Small Text
    • Justified Text
    • Underlined Text
    • and more!
  • Good help documentation
  • More than just finding issues
    • Page Outline
    • Disable Styles

Cons

  • Some issues require a manual check (potential for false positives)
  • Inline errors can be difficult to work with
    • Seems designed primarily for sighted users
    • Visually noisy by default; but can be customized
    • Sometimes hard to find issues on hidden page elements
