What Are Tables For?

  • Displaying tabular data
  • Not for layout

Anatomy of a table

<table>
<caption>Members of Luis's One-Man-Band</caption>

<tr>
<th scope="col">Name</th>
<th scope="col">Instrument</th>
</tr>

<tr>
<th scope="row">Luis</th>
<td>All of them</td>
</tr>

</table>
  • table Element
  • caption Element
  • tr Element: row of cells
  • th Element: header cell
  • td Element: data cell
  • scope Attribute
