Using radio buttons

  • Use labels
  • Use fieldsets
  • If doing more than "yes/no", consider using a select instead

Example radio Buttons

Are you awesome?

Code for Exampe radio Buttons

<fieldset>
<legend>Are you awesome?</legend>
<input id="yes" type="radio" name="awesome" value="yes">
<label for="yes">Yes</label>
<input id="very" type="radio" name="awesome" value="very">
<label for="very">Very Yes</label>
</fieldset>
