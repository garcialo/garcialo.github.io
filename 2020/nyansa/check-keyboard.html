  • General Rule: Everything that is mousable is keyboardable
  • Affects: people with mobility impairments, power users
  • How to check: don't use a pointing device

Keyboard usage

  • Tab: Next interactive item
  • Shift+Tab: Previous interactive item
  • Enter: Use a link (or button)
  • Space: Use a button (or scroll down)
  • Esc: Close flyouts, menus, modals, etc.
  • Arrow Keys: Scroll page or navigate in a widget
    • Selecting an item from menu
    • Changing selected radio button

Questions to ask yourself

  • Can I see where I am?
  • Can I get to all usable things?
  • Can I use all usable things?
  • Can I use the page without getting stuck?
