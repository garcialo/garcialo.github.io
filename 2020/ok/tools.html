Automated Testing Tools

There are paid solutions some of y'all may be using.

  • AMP - Level Access
  • ARC Monitoring - The Paciello Group
  • axe Monitor (formerly WorldSpace) - Deque
  • pa11y Dashboard - Team Pa11y
  • Pope Tech - Pope Tech
  • Tenon - Tenon

Note about pa11y

  • It's the only free/open source tool on this list
  • Can use axe instead of HTML Codesniffer
  • Requires some technical savvy to get up and running
  • Pa11y Dashboard on Github

The good and the bad

  • Work well for static, public-facing pages
    • if you have a lot of public sites or subsites there may be a place for one in your strategy
  • Not as good for testing...
    • flows
    • pages behind login
    • internal-only pages
  • Some tools give you a score
  • Can be pricey
    • less than a full-time employee
    • you will need someone to maintain it
  • Maintaining awareness for accessibility
