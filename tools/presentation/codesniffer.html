Luis's Take

  • Pretty interface
  • UI is pointer-centric
  • UI limits how much information it can provide
  • Consistent results between Chrome and Firefox
  • Doesn't work on some pages (ex. Twitter / Facebook)
  • Recommends fixes for poor contrast!
  • A lot of noise
    • Not always the best recommendation

  • 1 Error on Firefox - Color Contrast issue. #FFFFFF on #3399FF
  • Suggests changing background to 1076DC (great recommendation)
  • Interestingly, the same color issues exists on Chrome. I have no idea why it wasn't caught.
