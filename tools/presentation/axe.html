Luis's Take

  • Claims "No False Positives"
  • Consistent between Chrome and Firefox
  • No way to specify criteria by which to check
  • Could not get to work in Firefox Private Browsing Mode
  • Still early in development
  • Listen to a screen reader pronounce aXe (uh zee)
